About

Genre

Space Way is a browser game in which player controls a rocket in space

Technologies

This game is built using: JavaScript, Canvas, HTML, CSS

Sound effects

Sounds allow you to feel the mysterious atmosphere of space

Flat design

Desing is focused on a minimalist use of simple elements and colors

Controls

Move the rocket using the buttons WASD or Arrows.
On the way there will be a lot of barriers, such as meteorites and UFOs.
Shoot the barriers by pressing Space.

Screenshots

Click "Play" to start the game

Avoid meteorites or shoot them to get points

Every 20 seconds you will meet UFOS on the way

Click "Play again" to start the new game
